Linkr

This app is build with express.js for the backend, non-relational/NoSQL MongoDB as database, Object Data Modeling Mongoose library for schema-based solution to model application data, React, Redux, JavaScript for the frontend. Incorporated AWS S3 to store users profile picture to reduce load on the server.

In addition, the app uses Socket.io library to offer a real time messaging to users.
